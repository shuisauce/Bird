How we log birds

We keep a simple spreadsheet you can export as a CSV to update this website. Every row is a sighting. At minimum include the date and common name. If you add latitude and longitude, the map link is automatic.

  • Audubon page: we note the page number (and edition if needed) for quick reference.
  • Photos: put images into the images/ folder and add the filename in the photo column.
  • Map: either paste a Google Maps URL in maps_url or provide latitude and longitude.
